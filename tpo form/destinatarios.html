Qué elegir


Introducción a la programación

Aprenderás estructura de datos, conceptos básicos, operadores de lógica, iteración, condicionales, diagramas de flujo, paradigmas de programación y tipos de lenguajes; logrando al final de la cursada poder incorporarte en cualquier otro curso de programación avanzada habiendo consolidado conocimientos básicos.

Fullstack Developer

Aprenderás a programar en HTML, CSS, JavaScript, Python; utilizar GIT y Bootstrap. Serás capaz de utilizar conceptos de Scrum y Agile, aprenderás el uso y gestión de bases de datos con SQL. Logrando al final de la cursada desarrollarte como Frontend Developer, Backend Developer o ambos como Fullstack Developer.

Data Analytics

Aprenderás conceptos generales de bases de datos utilizando SQL, Podrás utilizar cualquier volumen de datos de manera eficiente para alcanzar tomas de decisiones informadas y confiables. Al terminar el curso podrás diseñar y desarrollar tableros de control utilizando Power BI, Power Query y Lenguaje M.

Data Science

Aprenderás conceptos teóricos y consideraciones técnicas para construir modelos de Machine Learning que puedan solucionar problemas reales de manera predictiva en distintos tipos de industrias. Utilizarás lenguaje Python para contruir programas que aprendan de la experiencia y puedan dar soporte en la toma de decisiones.
